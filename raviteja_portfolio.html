OCR Format Detection & Extraction Engine

A production-deployed document preprocessing pipeline that dramatically improved OCR accuracy for enterprise clients. The core innovation is a 4-model weighted scoring system that detects optimal document orientation before extraction — pushing confidence from ~60% to 88%+.

4 scoring models — OCR confidence, projection variance, structural line detection, aspect ratio — with weighted combination per document type (text / table / mixed)
Parallel processing via ThreadPoolExecutor for multi-model scoring without blocking the pipeline
